# PawlyApp API

API desarrollada con GraphQL y Apollo Server para la gestión de mascotas, incluyendo usuarios, perros, razas, ubicaciones, recordatorios, vacunas, citas veterinarias e historiales médicos.

Este proyecto fue desarrollado como proyecto final de la materia **Bases de Datos II**.

## Tecnologías utilizadas

- Node.js
- GraphQL
- Apollo Server
- MySQL
- Knex.js
- JWT
- Render

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm start
```

o

```bash
npm run dev
```

## Endpoint

```text
https://pawlyapp.onrender.com/
```
